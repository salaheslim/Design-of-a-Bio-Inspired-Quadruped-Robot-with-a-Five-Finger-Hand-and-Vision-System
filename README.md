# Design of a Bio-Inspired Quadruped Robot with a Five-Finger Hand and Vision System

This repository contains the CAD design files and documentation for a bio-inspired quadruped robot developed for the CIS7035 PRAC1 assessment.

## Project Overview
The robot combines:
- A dog-inspired quadruped base for stable locomotion
- A 6-DOF robotic arm
- A five-finger anthropomorphic hand
- A palm-mounted eye-in-hand vision system

The design focuses on mobility, stability, and dexterous manipulation in unstructured environments.
